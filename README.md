# Gold Price Forecasting & Risk Analytics (INR)

A Quantitative Research project to forecast 22K gold prices localized for the Indian market.

## Key Features
- **Forecasting:** Uses Holt-Winters Exponential Smoothing to capture seasonality in Indian gold demand.
- **Localized Pricing:** Integrates 6% Import Duty and 3% GST for retail-accurate valuation.
- **Risk Management:** Implements a 95% Confidence Interval (Z-score 1.96) to calculate Maximum Potential Price, aiding banks in LTV (Loan-to-Value) stress testing.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the model: `main.ipynb`
